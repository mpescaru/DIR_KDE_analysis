# DIR-KDE-analysis-

This code was used to calculate the %of DIRs (dyadic interval ratios) within set intervals corresponding to different small-integer ratio rhythms commonly observed in music (1:3, 1:2, 1:1, 2:1, 3:1). 
Ranges for each rhythm's intervals are the same as used in previous work (Roeske et al., 2020; De Gregorio et al., 2021). 

This project also includes functions for plotting DIR KDE curves and the use intervals for music and birdsong. 


Citations: 
DeGrazia, D., & Beauchamp, T. L. (2021). Beyond the 3 Rs to a More Comprehensive Framework of Principles for Animal Research Ethics. ILAR J, 60(3), 308-317. https://doi.org/10.1093/ilar/ilz011

Roeske, T. C., Tchernichovski, O., Poeppel, D., & Jacoby, N. (2020). Categorical Rhythms Are Shared between Songbirds and Humans. Curr Biol, 30(18), 3544-3555.e3546. https://doi.org/10.1016/j.cub.2020.06.072
